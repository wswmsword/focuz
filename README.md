# focuz

focuz 的特性：

- 支持嵌套结构聚焦；
- 延迟聚焦与失焦，指定初始焦点；
- 动态列表；
- 多入口、多出口；
- 具备开关功能的入口；
- 蒙层出口、<kbd>Esc</kbd> 出口；
- 支持调用式入口和出口；
- 焦点矫正、焦点保护；
- 导航序列列表、导航范围列表；
- 兼容鼠标聚焦。

基本入参结构：

```json
{
  "root": "#app",
  "entry": "#en1",
  "exit": "#ex1",
  "list": ["#i1", "#i2", {
    "entry": "#en2",
    "exit": "#ex2",
    "list": ["#i3"]
  }]
}
```

多组情况，类似如下：

```javascript
focuz(root, [{
  entry,
  exit,
  list
}, {
  entry,
  exit,
  list
}]);
```
